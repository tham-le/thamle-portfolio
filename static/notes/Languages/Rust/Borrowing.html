WTF is Borrowing in Rust?

Borrowing is Rust’s way of letting you use data without taking ownership. It’s like borrowing a book from a library - you can read it, but you have to return it.

There are two types of borrows:

  • Immutable borrow (&T) - Read-only access, many allowed simultaneously
  • Mutable borrow (&mut T) - Read-write access, only one at a time

The Borrow-Checker enforces these rules at compile time.

The Library Book Analogy

Borrowing Books

Immutable Borrow (Reading)

You check out a book to read:

  • Multiple people can read copies simultaneously
  • Nobody can write in the book while you’re reading
  • You must return it before the library closes (scope ends)

Example: Reference books, textbooks

Mutable Borrow (Writing)

You check out a book to write notes:

  • Only one person can have it at a time
  • Nobody else can read or write while you have it
  • You must return it before the library closes

Example: Guest book, sign-up sheet

Ownership (Buying)

You buy the book outright:

  • It’s yours forever
  • Do whatever you want with it
  • No returning required

The librarian (Borrow-Checker) enforces these rules at checkout time.

That’s borrowing - temporary access to data without taking ownership.

The Big Takeaway

Borrowing in a Nutshell

Borrowing lets you use data without taking ownership. Two types:

Immutable borrow (&T):

  • Read-only access
  • Many allowed simultaneously
  • Data cannot be modified

Mutable borrow (&mut T):

  • Read-write access
  • Only one at a time
  • No other borrows allowed

The Golden Rule: At any time, you can have either:

  • One mutable reference (&mut T), or
  • Any number of immutable references (&T)

Enforced by: Borrow-Checker at compile time (zero runtime cost)

Prevents: Data-Races, use-after-free, iterator invalidation

The rule: Borrowing is Rust’s way of safely sharing data. The Borrow-Checker enforces that readers don’t see partial writes and writers don’t conflict. It’s frustrating at first, but prevents entire classes of bugs that plague C++.
