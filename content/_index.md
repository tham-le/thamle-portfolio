## Hi, I'm Tham

I'm a systems engineer based in Paris, currently building a high-performance remote control SDK in Rust at [Kyber](https://gitlab.com/kyber.stream). Before that, I ported a C++ VPN SDK to WebAssembly at [Snowpack](https://snowpack.eu/).

My path to software started in climate science — FORTRAN simulations, IPCC-class models, large-scale data processing. That taught me to think in systems. Today I apply that to low-level development: input streaming, cross-platform networking, and security.

I also compete in [CTF challenges](/ctf/) with APT42 (42 Paris).
