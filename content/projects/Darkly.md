# Darkly — Web Security Challenges

A collection of web vulnerability exploits against a purposely vulnerable application. Each challenge documents the vulnerability, the exploit technique, and the fix.

## Vulnerabilities covered

- SQL injection (basic and blind)
- XSS (reflected and stored)
- Path traversal
- Cookie manipulation
- File upload exploits
- Hidden files and robots.txt analysis
- Open redirect
- Spoofing techniques
- Brute force

Each exploit is in its own directory with a script and writeup explaining the attack vector and how to defend against it.
